Cfields

jQuery plugin for creating fake HTML selects and checkboxes so there is the same functionality but more customisation

Making styling those pesky defaults a cinch!

cFields was created to make it easy to style custom fields which aren't customisable with standard CSS. The way it works is simple, we copy all of the content from the current field and create a HTML counterpart which the user can interact with and works in the same way as the real field, The difference is you can completely style it.

$(document).ready(function(){

    $('#selectField').cFields({
        label:true,
        speed: 'slow',
        animation: 'slide'
    });

});
